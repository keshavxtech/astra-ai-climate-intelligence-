
# 🌍 AstraAI - Climate & Health Intelligence System

AstraAI is an AI-powered system that provides real-time air quality insights and health recommendations.

## 🚀 Features
- Live AQI using API
- AI-based AQI prediction
- Health score calculation
- Activity recommendations
- Interactive dashboard

## 🛠 Tech Stack
- Python
- Streamlit
- Machine Learning (Random Forest)
- API Integration

## ▶️ Run Locally
```bash
pip install -r requirements.txt
streamlit run app.py
